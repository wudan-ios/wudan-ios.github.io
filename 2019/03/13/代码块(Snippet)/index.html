iOS代码块(Snippet)

发表于 2019-03-13 | 阅读次数: ℃
字数统计: 561 | 阅读时长 ≈ 2

代码块(Snippet)，方便了代码快速创建。

管理：在新版Xcode中管理页面在编译器的右上角

添加：选中代码鼠标右击

编辑

  1. Title：标题。
  2. Summary：描述文字。
  3. Platform：可以使用的平台（如iOS、Mac OS）。
  4. Language：可以在哪些语言中使用（如 Objective-C、Swift）。
  5. Completion Shortcut：快捷方式，以字母开头（支持少数符号，如@）。

  6. Completion Scopes：作用范围，一般写在正确的位置拖动即可，Xcode会自行选择好。

    属性

    @property (nonatomic, strong) <#ClassName#> *<#class#>; ///< <#注释#>
@property (nonatomic, assign) <#DataType#> <#name#>; ///< <#注释#>
@property (nonatomic, copy) <#ClassName#> *<#name#>; ///< <#注释#>
@property (nonatomic, weak) id<<#delegateOrDataSouce#>> <#delegateOrDataSouce#>; ///< <#注释#>
@property (nonatomic, weak) <#Class#> *<#name#>; ///< <#注释#>

    常用注释

  7. 顶部注释

    /** <#属性注释#> */
  8. 右侧注释
    ///< <#注释#>
  9. 简述顶部注释
    /** <#简要描述#>
* <#详细描述#>
*/
  10. Pragma
  • 有分割线
    #pragma mark <#mark#>
  • 无分割线
    #pragma mark - <#mark> 或 // MARK: <#mark#>
  1. Warning(警告)
    #warning <#message#>
  2. ???(不确定)
    // ???: <#???#>
  3. !!!(警告)
    // !!!: <#???#>

  4. FIXME(需要修改的地方)

    // FIXME: <#Fix详情#>
